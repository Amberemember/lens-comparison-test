# 百元 vs 万元？35mm 镜头成像对比测试

## 项目简介

本仓库用于托管文章 **《百元 vs 万元？探究 100 倍差价的相机镜头成像质量究竟有多大差别》** 及其配套的网页。

文章通过对比一支 **百元级国产工业改装镜头 TS35mm F1.6 II** 和一支 **万元级原厂索尼 35mm F1.4 GM（35GM）**，从多维度分析不同价位镜头在实际成像中的表现差异，包括：

- 像场覆盖
- 近摄表现
- 远景锐度（中心 / 边缘）
- 色散控制
- 抗眩光能力
- 虚化表现
- 焦外光斑特性

同时，项目中还包含一个简单的前端页面，用于交互式浏览各光圈、各镜头的对比样张。
