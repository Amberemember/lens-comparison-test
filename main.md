# 百元 vs 万元？探究100倍差价的相机镜头成像质量究竟有多大差别

本文为原创内容，**请勿转载**。

### 一、写在前面

本人非物理/光电专业学生，本文初衷为个人探索记录，并分享给8u们，仅代表个人观点，如有不实或者欠缺之处，欢迎批评指正。

本测试所有器材均为自费购买，未接受任何厂商赞助，也不构成任何购买建议。

近年来，随着国产镜头的崛起，越来越多的摄影爱好者开始关注性价比高的镜头产品。尤其是在定焦镜头领域，一些价格低廉的国产镜头凭借其出色的性价比，吸引了大量用户。然而，面对市场上动辄上万元的高端镜头，许多消费者不禁要问：价格差距如此之大，这些镜头在实际成像质量上到底有多大差别？

本文将通过对比一款售价不到百元的国产定焦镜头与售价上万元的索尼原厂旗舰镜头，深入探讨两者在实际拍摄中的表现差异。我们将从光学性能、成像质量、操作体验等多个方面进行详细分析，并通过实拍样片进行直观对比，帮助读者更好地理解不同价位镜头之间的差异。

如果您对前面的理论部分不感兴趣，建议直接跳转到实拍测试部分。


### 二、拍摄器材与拍摄设置

#### 1. 基本参数对比

本文将主要对比两款不同类型的镜头：国产 TS35mmF1.6II 手动定焦镜头（由工业镜头改装而来）和索尼 （Sony）35mmF1.4GM 定焦镜头。以下是两款镜头的主要参数对比：

| 商品名称  | TS3516II | SEL35F14GM    |
| --------- | -------- | ------------- |
| 售价      | **98元** | 11300元       |
| 画幅      | APS-C    | **全画幅**    |
| 镜头结构  | 4组/4片  | **10组/14片** |
| 最大光圈  | f/1.6    | **f/1.4**     |
| 光圈叶片  | 9        | **11**        |
| 最近对焦  | 0.45米   | **0.25米**    |
| 质量（g） | **115**  | 524           |
| 自动对焦  | 无       | **有**        |

> &#x2139;&#xfe0f; **品牌说明：**
> 
> 经过对市场上多款百元级公模镜头的初步调研与比较，天色（TIASE）品牌所提供的产品规格及光学参数相对最为完整。因此，本文暂以该品牌下的 TS35mmF1.6II 镜头作为此类镜头的代表性型号进行后续分析与讨论。

> &#128200; **MTF曲线**
> 
> MTF曲线可以标识一款镜头的理论光学性能。一般来说，MTF曲线整体越高，说明镜头光学分辨率越高；曲线整体越平直，说明中心和边缘画质差距越小；同颜色实线虚线贴合越近，说明镜头色散越小。
> 
> 关于MTF曲线的更多内容，可以参考以下视频：[【光学设计基础】镜头的MTF光学原理](https://www.bilibili.com/video/BV1B888zuEsA/?share_source=copy_web&vd_source=271aad73727f77d0778476fcede6d9a8)

从 MTF 中可以看到，TS35使用的工业镜头应该是针对小尺寸传感器进行优化，中心分辨率上表现尚可，但是距离镜头光学中心超过 9.6mm 之后，画质急剧下降（范围介于1英寸底到 m43 画幅之间，而 apsc 画幅的 14.1mm 甚至没有出现在 MTF 图内，说明这颗镜头设计之初，很可能并没有考虑到apsc画幅相机的使用）；而 35GM 的 MTF 图则在整个全画幅范围内，中心和边缘分辨率上都表现优秀，色散控制也较好。

当然，MTF曲线图仅仅是理论数据，实际成像效果还是需要通过实拍来进行验证。

可以看到，TS35mmF1.6II 虽然售价低廉，但意外的是它居然是全金属镜身，做工扎实，手感不错。~~（在此点名夸赞某大清光学厂）~~ 包装包含一个遮光罩，镜头本体和一个cctv口转索尼e口的转接环，整体的体积和重量都非常轻巧。而且最重要的是，这个镜头居然是有光学镀膜的！这在百元镜头中实属罕见。

在操作体验方面，TS35mmF1.6II 采用纯手动对焦与光圈调节设计。由于镜身体积较小，握持时缺乏足够的着力点，操作手感较为局促。此外，镜头与转接环之间采用螺纹连接，该连接结构在频繁手动对焦时容易发生松动，并可能因误触导致光圈值意外改变。另一方面，镜头遮光罩同样依赖螺纹安装，但在安装前若已盖上镜头盖，则会遮挡部分螺纹接口，致使遮光罩无法正常安装。

#### 5. 测试环境与拍摄设置

测试机身选用索尼a7r3，关闭镜头补偿，开启apsc模式，拍摄设置为M档，所有参数保持一致，拍摄时使用三脚架固定，仅更换镜头进行对比拍摄。

由于全画幅转换到apsc画幅会经历一定的像素裁切，转换后等效全幅焦距52.5mm，等效像素1800万。

> &#x2139;&#xfe0f; **测试说明：** 
> 
> 尽管1800万像素低于主流 apsc 相机的2400万像素，但在对比测试中完全足够。本次重点在于呈现两款镜头的相对差异，而非极限分辨率。

### 四、总结归纳

通过测试，我们可以发现，索尼 35GM 确实不负其万元旗舰镜头的称号，无论是在锐度、色散控制、抗眩光还是虚化效果等方面，都表现出色，整体成像质量优异。不过，TS35mmF1.6II 作为一款百元级别的镜头，其表现也超出了预期。在中心锐度、色散控制和抗眩光方面，TS35 展现出令人惊讶的实力。在工业高度发达的今天，光学设计和制造技术的进步使得低价镜头也能具备一定的成像质量。

这背后折射出一个更深层的逻辑：在镜头这类精密光学产品上，价格与性能并非线性关系，而是显著的边际递减。因此，选择镜头的本质，是寻找个人需求、创作阶段与预算约束之间的平衡点。无论是用百元镜头探索世界，还是用万元工具追逐极致，都值得尊重。真正重要的是，我们应透过镜片去理解光，通过取景器去看见美——毕竟，驱动相机的那份观察与思考，远比相机前端的任何一枚镜头都更为珍贵。
